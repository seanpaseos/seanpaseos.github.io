Web Application Overview

This web application is designed to manage and streamline vaccine inventory, patient records, supply orders, and staff administration for a healthcare facility. It ensures efficient tracking of vaccines, facilitates order management, and maintains accurate records of administered doses.

Key Features
  • User Authentication: Role-based access for Doctors and Assistants.
  • Patient Management: Record and track patient vaccinations.
  • Vaccine Inventory: Monitor stock levels, purchases, and expirations.
  • Order Management: Manage vaccine supply orders from suppliers.
  • Stock Management: Track vaccine distribution and assistant handling.

Troubleshooting Errors
  • Unable to Start XAMPP: Ensure Apache and MySQL are not blocked by a firewall or another application.
  • Check Port Usage: Run the following commands to check if ports 80 (Apache) and 3306 (MySQL) are in use: 
    netstat -ano | findstr :80
          netstat -ano | findstr :3306
    If another application is using these ports, either stop that process or change XAMPP's port settings.
  • Port 3000 is Already in Use: To free up port 3000, follow these steps:
    • Find the Process ID (PID) using: 
      netstat -ano | findstr :3000
    • This will return a line like: 
      TCP    0.0.0.0:3000    0.0.0.0:0   LISTENING  12345
      Here, 12345 is the PID.
    • Kill the process with: 
      taskkill /PID 12345 /F
      Replace 12345 with the actual PID you found.
    • Go back to the Terminal 
      Input : node server.js
